cksm_val(引数１, 引数２)¶

Meridim配列のチェックサムを算出します。
引数1にはMeridim配列、引数2には配列の長さを設定します。
Meridim配列の末尾-1までの数値を合計し、最後にビット反転しShort型に格納したものをチェックサム値として返します。

なお、このチェックサムはMeridim配列の末尾に格納します。 


// +----------------------------------------------------------------------
// | func name : cksm_val(short arr[], int len)
// +----------------------------------------------------------------------
// | function  : calculate checksum of Meridim
// | argument1 : short, Meridim array
// | argument2 : int,   Length of array
// | return    : short, checksum value 
// +----------------------------------------------------------------------
short Meridian::cksm_val(short arr[], int len)
{
    int cksm = 0;
    for (int i = 0; i < len - 1; i++)
    {
        cksm += int(arr[i]);
    }
    return short(~cksm);
}
